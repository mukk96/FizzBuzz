# FizzBuzz
Fizz Buzz プログラム集

## 概要
FizzBuzz問題の解答を示したプログラム集です。

## 特徴
いずれの言語におけるコードも、以下を意識して記述しています。

* if（分岐）, for（繰り返し）という基本ロジックが理解できていること。（FizzBuzz問題の基本）
* 関数定義を含めること。
* コードが読みやすいこと。難解な解法は用いない。

このようにした理由は、大人数での開発プロジェクトでは上記のようなコードが求められることが多いとされるからです。

## 動作

結果に改行をつけて標準出力に出力します。

## インストール

（動作確認はいずれも WSL で行っています）

$ make

C と Java の実行ファイルが生成されます。

## 使用法

$ make test

実行ファイルを実行して回答をテキストファイルにいったん保存し、それらとfizzbuzzの正答ファイル（fizzbuzz.answer）との比較を行って正答かどうかを表示します。

$ make clean

実行結果のテキストファイルなどを削除します。
